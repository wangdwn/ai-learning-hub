AI基础认知

第1课：什么是人工智能？

入门级 10分钟阅读

通俗理解

人工智能（AI）就像是给计算机装上了一个"大脑"，让它能够：

  • 理解你说的话 - 就像和朋友聊天一样自然
  • 帮你写东西 - 邮件、报告、文案都能写
  • 分析问题 - 给你建议、帮你做决策
  • 处理重复工作 - 节省你的时间和精力
生活比喻：AI就像是一个非常聪明、从不疲倦的助手，它读过互联网上几乎所有的书籍和资料，可以随时为你提供帮助。

AI能做什么？

  1. 文本处理 - 写作、翻译、总结、润色
  2. 数据分析 - 解读数据、制作图表、发现规律
  3. 编程辅助 - 写代码、找bug、解释技术概念
  4. 创意激发 - 头脑风暴、提供灵感、设计方案
  5. 学习辅导 - 解释复杂概念、制定学习计划

AI不能做什么？

  • 替代人类的创造力和情感
  • 做出涉及伦理的最终决策
  • 获取实时信息（除非特别配置）
  • 完全理解特定公司的内部业务逻辑
动手练习：打开任意一个AI工具（如Claude、ChatGPT），问它："用简单的话解释什么是人工智能，并举3个生活中的例子"

第3课：如何与AI有效对话

入门级 必学

黄金法则：清晰、具体、有背景

❌ 不好的提问：

"帮我写个邮件"

✅ 好的提问：

"我需要给客户写一封跟进邮件。背景：我们上周提交了项目方案，客户说需要考虑。请帮我写一封礼貌、专业的跟进邮件，询问他们的决定，并表示愿意解答任何问题。语气要友好但不失专业。"

提问公式：背景 + 任务 + 要求 + 格式

  1. 背景 - 告诉AI你现在的情况
  2. 任务 - 明确你要AI做什么
  3. 要求 - 说明具体要求和限制
  4. 格式 - 指定输出的形式
实战练习：用上面的公式，让AI帮你写一份本周工作总结。包含：你做了什么、遇到什么困难、下周计划。

第1课：提示词工程基础

入门级 核心技能

什么是提示词工程？

提示词工程（Prompt Engineering）就是学会如何给AI下指令，让它给你最好的回答。就像学会和同事有效沟通一样重要。

五大提示词技巧

1. 角色设定法

// 让AI扮演特定角色 "你是一位经验丰富的产品经理，请帮我..." "假设你是一位资深HR，请给我建议..." "请扮演一位耐心的老师，向我解释..."

2. 分步骤法

// 将复杂任务分解 "请按以下步骤帮我： 1. 先分析这个问题的原因 2. 然后列出3个解决方案 3. 最后比较各方案的优缺点"

3. 示例法（Few-shot）

// 给AI例子，它学得更快 "请按照下面的风格回复客户： 例子1： 客户：你们的产品太贵了 回复：理解您的考虑...（展示同理心+解释价值） 现在请回复：客户说发货太慢了"

4. 约束条件法

// 明确告诉AI限制条件 "请用300字以内总结" "请避免使用专业术语" "请用 bullet points 格式输出" "目标受众是小学生，请用简单语言"

5. 迭代优化法

// 不满意就继续调整 第一次："帮我写个产品介绍" 第二次："太长了，缩短到100字" 第三次："再加一些数据支持" 第四次："语气再活泼一些"
秘诀：把AI当成一个聪明但需要你指导的实习生。你越清楚地说明需求，结果就越好。

  • 分析项目结构和文件

// 进入项目目录后启动 claude // 常用指令 /help // 查看帮助 /clear // 清空对话 /compact // 压缩对话历史

逻辑思维训练

第1课：结构化思维

进阶级 思维方法

什么是结构化思维？

结构化思维就是把复杂的问题分解成清晰、有条理的部分。就像整理衣柜一样，把东西分类放好，需要时就能快速找到。

MECE原则：不重不漏

MECE = Mutually Exclusive, Collectively Exhaustive（相互独立，完全穷尽）

// 不好的分类：有重叠 人群分类：男人、女人、小孩、老人 // 问题：小孩可以是男孩或女孩，老人也有男女 // 好的分类：MECE 人群分类： - 按性别：男性、女性 - 按年龄：儿童、青年、中年、老年

常用思维框架

1. 5W2H分析法

  • What - 是什么？要做什么？
  • Why - 为什么？目的是什么？
  • Who - 谁？涉及哪些人？
  • When - 何时？时间安排？
  • Where - 何地？在哪里做？
  • How - 怎么做？方法是什么？
  • How much - 多少？预算/成本？

2. SWOT分析

S - 优势

内部的有利因素

W - 劣势

内部的不利因素

O - 机会

外部的好机会

T - 威胁

外部的挑战

练习：用5W2H分析法，向AI描述你最近要完成的一个工作任务，看看AI能否帮你梳理得更清晰。

第2课：批判性思维

高级 必备能力

AI时代的批判性思维

AI可以给我们答案，但它不一定总是对的。批判性思维帮助我们：

  • 判断AI回答的质量
  • 发现潜在的错误或偏见
  • 结合实际情况做决策

验证AI回答的 checklist

  1. 事实核查 - 关键数据是否准确？
  2. 逻辑检查 - 推理过程是否合理？
  3. 时效性 - 信息是否过时？
  4. 偏见识别 - 是否存在明显偏见？
  5. 适用性 - 是否适合我的具体情况？

如何向AI提问以获得更好答案

// 让AI展示思考过程 "请一步一步地思考这个问题" "请解释你的推理过程" // 让AI考虑不同角度 "请从正反两方面分析这个问题" "有哪些反对这个观点的理由？" // 让AI承认不确定性 "如果你不确定，请告诉我" "这个答案的可靠程度如何？"
重要提醒：对于重要决策，永远不要把AI的回答作为唯一依据。用AI获取思路和参考，但最终判断权在你。

第3课：与AI协作解决问题

进阶级 实战技巧

人机协作的最佳模式

把AI当作你的思维伙伴，而不是答案机器。最佳协作流程：

  1. 你定义问题 - 清楚地描述你要解决什么
  2. AI提供思路 - 让AI给出可能的方向
  3. 你筛选判断 - 结合实际情况选择可行方案
  4. AI深化细节 - 对选定方案进行细化
  5. 你执行验证 - 实施方案并检验效果

实战：用AI解决工作难题

// 第一步：描述问题 "我遇到一个工作问题：[具体问题描述] 我已经尝试了：[你做过什么] 限制条件：[时间/资源/人员等限制]" // 第二步：要求多角度方案 "请提供3个不同角度的解决方案， 并分析每个方案的优缺点和所需资源" // 第三步：深入最佳方案 "我选择方案X，请帮我： 1. 制定详细的执行步骤 2. 识别可能的风险 3. 准备应对预案"
思维实验：想一个你目前工作中的真实问题，按照上面的流程和AI对话，看看能否找到新的解决思路。

团队协作能力

第1课：团队中的AI使用规范

入门级 必读

为什么要建立使用规范？

当团队一起使用AI时，需要明确的规则来确保：

  • 信息安全和保密
  • 工作质量的一致性
  • 团队成员之间的有效协作
  • 避免过度依赖AI

建议的团队规范框架

1. 信息安全红线

绝不要输入AI的内容：
  • 客户敏感信息（身份证号、银行卡号等）
  • 公司核心商业机密
  • 未公开的产品计划或财务数据
  • 员工个人隐私信息

2. 使用场景白名单

  • ✅ 可以：通用知识查询、文案草稿、代码辅助、学习资料整理
  • ❌ 避免：涉及敏感数据的分析、最终决策、对外发布的最终版本

3. 质量检查流程

  1. AI生成内容 → 人工审核 → 修改完善 → 最终确认
  2. 重要文档必须人工复核事实和数据
  3. 对外发布的材料需要负责人签字确认

团队提示词模板共享

建议团队建立共享的提示词库，包含：

  • 常用文案模板（邮件、报告、通知）
  • 内部术语解释
  • 公司风格指南
  • 常见问题解决方案
