🔑 如何获取自己的 DeepSeek API Key

全程只需5分钟，完全免费，注册即送额度 免费

1

打开 DeepSeek 开放平台

点击下方链接，用手机号或邮箱注册账号：
👉 platform.deepseek.com

💡 已有DeepSeek账号（app或网页版）可以直接登录，不需要重新注册。
2

进入 API Keys 页面

登录后，点击左侧菜单的 「API Keys」，然后点击右上角 「创建 API Key」 按钮。

📌 左侧菜单 → API Keys → 创建 API Key
3

给 Key 起个名字，点击创建

名字随便填，比如"学习平台"，然后点击 「确认」。

⚠️ Key 只显示一次！创建后立即复制保存好，关闭弹窗就看不到了。
4

复制 Key，填入学习平台

复制以 sk- 开头的那串字符，回到学习平台：

点击 「首次使用需要配置API密钥」 → 选择 DeepSeek → 粘贴进去 → 点 「保存」。

🔒 Key 只保存在你自己的浏览器里，不会上传到任何服务器，放心使用。
5

开始提问！

配置完成后，在输入框里输入你的问题，点击 「向AI提问」 即可。

💰 注册送的免费额度够日常学习使用，用完后按需充值，价格非常低。

🎉 配置完成！

遇到问题？把报错截图发给管理员，或者重新检查 Key 是否完整复制（以 sk- 开头）。
